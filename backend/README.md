# Be The Hero - Backend

A backend developed with Node.js, express.js, Knex.js and SQLite.

---
## Install

    $ git clone https://github.com/pedroigorjs/bethehero.git
    $ cd backend
    $ yarn install

## Configure app

Open `.env.example` and copy to `.env` then edit it with your settings. You will need:

- PORT
- HOST
- DB_HOST
- DB_NAME
- DB_USERNAME
- DB_PASSWORD

## Running the project

    $ yarn start:dev
